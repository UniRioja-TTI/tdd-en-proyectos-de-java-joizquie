# Taller Transversal I: Práctica III - TDD en Java

Este proyecto forma parte de la asignatura **Taller Transversal I** de la **Universidad de La Rioja**. Su objetivo principal es analizar la estructura estándar de un proyecto Java y practicar la metodología de Desarrollo Guiado por Pruebas (TDD).

## 🛠️ Tecnologías utilizadas

* **Java 21**: Versión del lenguaje configurada en el compilador.
* **Maven**: Herramienta de gestión de proyectos y dependencias.
* **JUnit Jupiter (5.10.0)**: Framework utilizado para la ejecución de pruebas unitarias.

## 📂 Estructura del proyecto

El proyecto sigue la estructura estándar de Maven:
* `src/main`: Contiene el código fuente de la aplicación y recursos de configuración (`application.properties`).
* `src/test`: Ubicación de las clases de prueba unitarias.
* `pom.xml`: Archivo de configuración de Maven donde se definen dependencias y plugins como `maven-surefire-plugin`.
* `.editorconfig`: Archivo para mantener la consistencia de estilos (4 espacios, codificación UTF-8) entre diferentes editores.
